# 🧠 Fake News & AI Content Detector

A machine learning + Streamlit web application that detects whether a given text is **Fake News or Real News**, and also classifies whether content is **AI-generated or Human-written**.

---

## 🚀 Features
- Fake News detection using TF-IDF + Logistic Regression
- AI vs Human text analysis using heuristic scoring
- Image analysis with OpenCV (face detection + basic AI check)
- Interactive Streamlit UI
- Real-time predictions

---

## 🛠️ Technologies Used
- Python
- Streamlit
- Scikit-learn
- NumPy
- Pillow
- OpenCV
